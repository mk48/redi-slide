# Executes a function for each array element

```javascript
let numbers = [1, 2, 3, 4];

let execute = function(n) {
  console.log(n);
}

numbers.forEach(execute);

// Output:
// 1
// 2
// 3
// 4
```   

# Move function inside forEach()

```javascript
let numbers = [1, 2, 3, 4];

numbers.forEach(
  function(n) {
    console.log(n);
  }
);

```
