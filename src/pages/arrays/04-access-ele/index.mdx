**Use square brackets with index number (starting from 0)**

```javascript
let pets = ["dog", "cat", "hamster", "fish"];

console.log(pets[0]); // "dog"
console.log(pets[2]); // "hamster"

// Last element
console.log(pets[pets.length - 1]); // "fish"
```
